# Albert Knaus – Künstler-Webseite

Responsive, statische Portfolio-Webseite für GitHub Pages.

Willkommen in meiner Welt der modernen Kunst. Entdecken Sie einzigartige,
handgefertigte Acrylbilder, abstrakte Reliefkunstwerke und außergewöhnliche
Skulpturen. Jedes meiner Werke ist ein Unikat und verbindet ausdrucksstarke
Farben, Strukturen und moderne Eleganz.

## Vor der Veröffentlichung ergänzen

1. Die sechs Platzhalterflächen in `index.html` durch echte Werkfotos ersetzen.
2. E-Mail-Adresse und vollständige ladungsfähige Anschrift im Impressum ergänzen.
3. Datenschutzerklärung rechtlich prüfen und vervollständigen.
4. Optional Verkaufslinks zu eBay, Vinted oder Instagram ergänzen.

## Veröffentlichung auf GitHub Pages

Das Repository muss `albertknaus4-design.github.io` heißen. Danach unter
`Settings → Pages → Deploy from a branch` den Branch `main` und den Ordner `/ (root)` auswählen.
